# simple-cmake
Simple C++ program with a library, built with cmake.
WASM playground with webpack environment and typescript.


## Standalone environment:

### Build C:
`./destroy-rebuild-standalone.sh`


### Run:
`./build/bin/main`


## WASM environment:

### Set up playground:

`cd wasm_playground`
`npm i`


### Build C:

`./destroy-rebuild-standalone.sh`


### Run playground:

- `cd wasm_playground`
- `npm start`
- visit localhost:8080
